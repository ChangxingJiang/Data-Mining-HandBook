# 案例1：LPL比赛结果预测

这里案例使用LPL比赛的历史数据，尝试对进行的LPL比赛结果进行预测。具体地，包括数据采集、数据清洗、数据分析三个部分。

预测包括：

* 依据当前经济曲线预测比赛结果
* 依据英雄选择预测比赛结果
* 依据队伍预测比赛结果
* 依据选手预测比赛结果
* 依据选手预测英雄选择
* 依据队伍预测比赛时长
* 依据选手预测个人KDA
* 根据前15分钟的经济曲线预测比赛时长
* ……
